## Description

[Nest](https://github.com/nestjs/nest) TypeScript user CRUD application.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
